# PROJETO — CENTRAL DE CARTEIRA FENIÉ

## Objetivo

Criar um sistema interno para gerenciamento de carteira comercial da Fenié PRO.

O sistema NÃO será um SaaS para venda.
Será uma plataforma interna para:
- acompanhamento de clientes
- reativação de carteira
- controle de follow-up
- gestão da equipe comercial
- acompanhamento de metas
- visualização de oportunidades

---

## Referência visual

Usar como inspiração visual o sistema MERCOS.

Características:
- layout claro
- sistema corporativo
- visual leve
- menu lateral esquerdo
- dashboards simples
- tabelas limpas
- foco em produtividade

Evitar visual escuro/luxuoso do protótipo atual.

## Regras de negócio

### Classificação automática

0-30 dias = saudável
31-60 dias = atenção
61-90 dias = risco
90+ dias = inativo antigo
