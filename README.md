# Central de Carteira Fenie

Sistema interno para gestão da carteira comercial da Fenie, com frontend
Next.js e banco/autenticação no Supabase.

## Estrutura

- `frontend/`: aplicação Next.js.
- `backend/supabase/`: migrations e configuração do Supabase.
- `docs/`: documentação funcional, técnica, operacional e de deploy.
- `tools/`: scripts para ambiente local e migração SaaS.

## Desenvolvimento local

Consulte:

- `LOCAL_INSTALL.md`
- `LOCAL_RUNBOOK.md`
- `docs/MIGRACAO_SAAS_INTERNO.md`

Arquivos `.env`, backups e planilhas reais não devem ser enviados ao Git.
